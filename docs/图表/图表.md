## 🎯 概述

大模型可以通过多种方式生成图表：
- **Mermaid语法** - 流程图、时序图、甘特图
- **SVG绘制** - 自定义矢量图形
- **Python代码** - matplotlib、plotly、seaborn
- **HTML/CSS** - 交互式图表
- **代码库调用** - echarts、d3.js、chart.js
