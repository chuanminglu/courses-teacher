## 🚀 测试工作流

### 步骤1: 生成布局骨架

```markdown
# 在Copilot Chat / Claude中

1. 打开 P01-布局骨架生成提示词-通用版.md
2. 将 P00-布局信息提取结果.md 中的 Part 1 粘贴到"材料1"位置
3. 执行提示词，获取 DashboardLayout.vue 等文件
```

### 步骤2: 生成设计系统

```markdown
1. 打开 P02-设计系统生成提示词-通用版.md
2. 将 Part 2 粘贴到"材料1"位置
3. 执行提示词，获取 variables.scss、theme.ts 等文件
```

### 步骤3: 生成图表组件

根据 Part 3 的组件清单，选择对应的提示词：

| 组件 | 提示词 |
|------|--------|
| 综合指标仪表盘 | P03-仪表盘组件 |
| 资源汇总环形图 | P04-环形图组件 |
| 资源统计柱状图 | P05-柱状图组件 |
| 运行状态折线图 | P06-折线图组件 |
| 系统关联图 | P07-关系图组件 |
| 用户运营地图 | P08-地图组件 |
| 各类卡片 | P09-数据卡片组件 |
| 各类列表 | P10-列表表格组件 |

### 步骤4: 组装页面

```markdown
1. 打开 P11-组件组装提示词-通用版.md
2. 将 Part 4 的映射表粘贴到输入材料
3. 执行提示词，获取完整页面组装代码
```

### 步骤5: 数据对接

```markdown
1. 打开 P12-数据对接提示词-通用版.md
2. 提供后端API文档
3. 执行提示词，获取数据层代码
```
