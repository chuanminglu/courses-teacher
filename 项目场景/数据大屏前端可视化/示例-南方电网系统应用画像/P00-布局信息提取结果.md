## 📋 使用说明

### 快速使用

1. **生成布局骨架**: 复制 Part 1 → 粘贴到 P01 提示词的"材料1"
2. **生成设计系统**: 复制 Part 2 → 粘贴到 P02 提示词的"材料1"
3. **生成图表组件**: 根据 Part 3 的"提示词映射"选择对应的 P03-P10
4. **组装完整页面**: 复制 Part 4 → 粘贴到 P11 提示词的映射表

### 执行顺序

```
P01 布局骨架 → P02 设计系统 → P03-P10 各组件 → P11 组装 → P12 数据对接
     ↑              ↑              ↑              ↑           ↑
   Part 1        Part 2        Part 3        Part 4      API文档
```

### 技术栈要求

```markdown
- 框架：Vue 3 + Composition API
- 样式：SCSS + CSS Variables
- 图表：ECharts 5
- 构建：Vite
- 规范：TypeScript
```
