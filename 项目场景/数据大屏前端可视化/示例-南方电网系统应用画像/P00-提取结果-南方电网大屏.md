## 📝 生成P07地图组件的推荐配置

虽然当前页面没有地图，但如果要在"用户运营分析"模块中添加地图，以下是推荐的配置：

```markdown
【组件名称】用户地域分布地图

【地图范围】
- 地图类型：中国地图
- 具体区域：全国（省级）
- GeoJSON来源：阿里云DataV https://geo.datav.aliyun.com/areas_v3/bound/100000_full.json

【视觉效果】
- 尺寸：382px × 200px
- 区域填充：渐变色，范围 #0d2847 (低) - #00d4ff (高)
- 边界线：颜色 #1a3a5c，宽度 1px
- 背景色：transparent

【数据展示】
- 数据类型：区域着色 (热力图效果)
- 标签显示：隐藏 (hover时显示省份名称)

【交互效果】
- Hover：区域高亮 + 显示Tooltip (省份名 + 用户数)
- 点击：触发事件 (emit province-click)
- 缩放：不支持
- 拖拽：不支持

【数据结构】
interface UserDistribution {
  name: string    // "广东省"
  value: number   // 12580
}
```
