# P02-设计系统生成提示词（通用版）

> **📌 用途**: 生成数据大屏的完整设计系统（Design System），包含色彩、字体、间距、阴影等Design Token。

---

你是一位资深UI设计系统架构师，拥有8年设计系统建设经验，擅长：

### 任务上下文

设计系统将作为所有大屏组件的基础，确保视觉一致性。需要同时支持SCSS变量和CSS自定义属性，便于组件开发和运行时主题切换。

## 🎯 G - 目标与意图

### 核心目标

建立**完整、一致、可扩展**的设计系统，确保所有组件视觉风格统一。

### 具体目标

1. **Token化**: 所有设计决策转化为可复用的Design Token
2. **语义化**: Token命名反映用途而非具体值（如 `--color-danger` 而非 `--color-red`）
3. **可扩展**: 支持新增主题变体（如高对比度模式）
4. **工程化**: 提供SCSS函数/Mixin简化开发

### 业务价值

- **为设计师**: 提供明确的设计规范，减少设计走查时间
- **为开发者**: 提供即用的样式工具，避免重复编写基础样式
- **为产品一致性**: 确保跨组件、跨页面的视觉一致性

### 成功标准

- ✅ 色彩系统包含语义化命名（primary/success/warning/danger）
- ✅ 提供至少5个常用Mixin（flex-center、ellipsis、scrollbar等）
- ✅ 所有Token有JSDoc注释说明用途
- ✅ 包含使用示例文档

## 📤 O - 输出要求

### 1. 输出结构

```
src/styles/
├── tokens/
│   ├── colors.scss        # 色彩Token
│   ├── typography.scss    # 字体Token
│   ├── spacing.scss       # 间距Token
│   └── effects.scss       # 阴影/动画Token
├── mixins/
│   ├── layout.scss        # 布局Mixin
│   ├── typography.scss    # 文字Mixin
│   └── effects.scss       # 效果Mixin
├── base/
│   ├── reset.scss         # 样式重置
│   └── global.scss        # 全局基础样式
└── index.scss             # 统一入口
```

### 2. 质量要求

- 每个Token必须有注释说明用途和使用场景
- Mixin必须有参数说明和使用示例
- 色彩对比度符合WCAG AA标准（文字:背景 ≥ 4.5:1）

### 3. 格式规范

- SCSS变量使用 `$` 前缀
- CSS变量使用 `--` 前缀
- Mixin使用 `@mixin` 定义，`@include` 调用

### 4. 特别说明

#### 双轨制变量系统

同时输出SCSS变量和CSS变量，支持编译时和运行时两种使用方式：

```scss
// SCSS变量（编译时）
$color-primary: #00d4ff;

// CSS变量（运行时）
:root {
  --color-primary: #{$color-primary};
}
```

#### 常用Mixin清单

至少包含以下Mixin：

| Mixin名称 | 用途 |
|----------|------|
| `flex-center` | Flex居中布局 |
| `flex-between` | Flex两端对齐 |
| `ellipsis` | 单行文字省略 |
| `multi-ellipsis($lines)` | 多行文字省略 |
| `scrollbar` | 自定义滚动条样式 |
| `card-base` | 卡片基础样式 |
| `glow-effect` | 发光效果 |

## 📝 使用示例

将上述"输入材料"中的占位符替换为实际项目信息，例如：

```
【色彩系统】
- 背景色：#0d1b2a（主背景）、#1b2838（卡片背景）、#243447（悬浮背景）
- 主色：#00d4ff（青色）、#0066ff（蓝色）
- 辅助色：#00ff88（绿色/成功）、#ff6b6b（红色/告警）、#ffd93d（黄色/警告）
- 文字色：#ffffff（主文字）、#8899aa（次要文字）、#556677（禁用文字）
- 边框色：rgba(0, 212, 255, 0.3)（默认）、rgba(0, 212, 255, 0.6)（悬浮）
...
```

然后将完整提示词发送给AI即可。
