## 📋 T - 任务描述

基于以下需求，生成数据卡片组件系列。

### 输入材料

#### 材料1：组件需求

```
【组件系列】数据卡片（DataCard）

【变体类型】（选择需要的变体）

1. 基础卡片（BasicCard）
   - 结构：标题 + 数值 + 趋势箭头
   - 适用：展示单一KPI指标
   - 示例：工单数 182 ↑12%

2. 图标卡片（IconCard）
   - 结构：图标 + 标题 + 数值
   - 适用：带图标的指标展示
   - 示例：🔔 紧急告警 36

3. 进度卡片（ProgressCard）
   - 结构：标题 + 数值 + 进度条
   - 适用：完成率/占用率展示
   - 示例：作业完成率 98% [████████░░]

4. 状态卡片（StatusCard）
   - 结构：状态指示灯 + 标题 + 描述
   - 适用：系统状态展示
   - 示例：🟢 系统状态：正常运行

【通用样式】
- 背景：{卡片背景色，建议使用CSS变量}
- 边框：{边框样式}
- 圆角：{圆角大小}
- 内边距：{内边距}
- 数字字体：{数字专用字体}

【动画效果】
- 数字变化：{滚动动画/闪烁/无}
- Hover：{边框发光/上浮/无}
- 状态变化：{平滑过渡/无}

【数据结构】
// BasicCard
{ title: string, value: number, trend?: number, trendDirection?: 'up' | 'down' }

// IconCard
{ icon: string, title: string, value: number, status?: 'normal' | 'warning' | 'danger' }

// ProgressCard
{ title: string, value: number, max: number, unit?: string }

// StatusCard
{ status: 'success' | 'warning' | 'danger' | 'info', title: string, description: string }
```

#### 材料2：技术要求

```
- 框架：{Vue 3/React 18}
- 样式：{SCSS/Tailwind}，使用设计系统CSS变量
- 图标库：{Iconify/RemixIcon/FontAwesome/自定义}
```

## 🎯 G - 目标与意图

### 核心目标

生成一套**统一风格、高度可配置**的数据卡片组件，支持大屏各区块的数据展示需求。

### 具体目标

1. **风格统一**: 所有变体共享基础样式，视觉一致
2. **配置灵活**: 通过Props支持各种展示形态
3. **动效精致**: 数字滚动、Hover反馈增强交互体验
4. **响应式**: 支持不同容器尺寸自适应

### 业务价值

- **为业务人员**: 快速获取关键指标，支持决策
- **为开发团队**: 可复用组件系列，减少重复开发
- **为设计一致性**: 统一的卡片风格，提升整体美观

### 成功标准

- ✅ 4种变体组件API设计一致
- ✅ 支持深色/浅色主题自动切换
- ✅ 数字滚动动画流畅
- ✅ 代码包含完整TypeScript类型定义

- BEM命名规范
- 避免魔法数字

### 3. 格式规范

- Vue SFC格式
- SCSS嵌套不超过3层

### 4. 特别说明

#### 数字滚动实现

```typescript
import { useNumberAnimation } from '@/composables/useNumberAnimation'

const { displayValue } = useNumberAnimation(
  () => props.value,
  { duration: 500, easing: 'easeOutCubic' }
)
```

#### 趋势箭头配色

```typescript
const trendColor = computed(() => {
  if (!props.trend) return ''
  return props.trend > 0 
    ? 'var(--color-success)' 
    : 'var(--color-danger)'
})
```
